卒業論文『効率とシステムサイズの関係に対する確率モデルによる考察』

背景

  • アロメトリー則(\(E\sim M^{3/4}\))
  • ホヤの実験によるシステムサイズと代謝率の関係
  • 会議やグループワークの参加人数と効率の関係

  • 一般化

    系のシステムサイズが大きくなると、その特徴量がシステムサイズに対して比例して大きくはならず、相互作用などによって期それより小さくなるような系

アプローチ

  • 会議をイメージしながらも、より一般的、抽象的なモデルを作成
  • モデルに関して解析的な計算、数値シミュレーション
  • 特徴量のシステムサイズに対する応答を調べる

モデルの作成

設定

  • はじめに一つの議題が与えられる
  • システムサイズは会議の参加者の人数\(N\)とする
  • 参加者\(X_{1}, \cdots , X_{N}\)
  • 参加者はそれまでに提出された意見に関連した自分の意見を述べる
  • 発言の長さは参加者の数や意見には影響されないとする(離散時間\(k\)の導入)
  • ある時刻\(k\)に発言するのは一人のみであるとする
  • 全体を統括する役割を持った参加者は考えない
  • 発言はそれを特徴づける\(a\)(\(a<+\infty\))個のパラメータの値で表せる
  • 各離散時刻\(k\)になされた発言\(x_{k}\)は\(\mathbb{R}^{a}\)上の有界集合\(\Omega\subset \mathbb{R}^{a}\)における1点
  • 時刻\(k\)を1進めるごとに1つの点\(x_{k}\)を選択し、この試行を時刻\(K\)まで続ける

意見の選ばれ方と選ばれた意見の位置関係によって決められるネットワークの性質によって会議の効率を考える

用語

  • 会議 \(\rightarrow \)試行
  • 意見 \(\rightarrow \)点(\(x\))
  • 参加者 \(\rightarrow \)集合\(\displaystyle X_{i} \subset X \left( = \bigcup _{i=1}^{N}X_{i}\right)\)

点の選び方とネットワークの構成方法をいくつか仮定してモデルを作成。
それぞれにおける特徴量を計算し、システムサイズ\(N\)との関係を考えるのにふさわしいはモデルはどれかを考察する

解析

1. \(X_{i}\)を選ぶ確率が等しく、点\(x\)が\([0,1]\)から一様に選ばれる場合

  • 点の選択規則:
    • \([0,1]\)の範囲から一様に選ばれる
  • エッジの生成規則:
    • 選択された点が閾値\(r\)により定まる領域内にあるとき、そのノード間にエッジを張る

時刻\(k+1\)に点\(x_{k+1}\)が選ばれた時、\(r\)によって定まる領域

\[[\max(0,x-r), \min(x+r,1)]\]

に入る点の個数\(Y\)は、それまでに選択された点の数が\(k\)個あることと、\(x_{k+1}\)の大きさによってその中に点が存在する確率が異なることを考慮に入れて計算すると、確率変数\(Y\)は以下の分布に従うことが分かる。

\[P[Y=y] = \ _{k}C_{y}p(x_{k+1}, r)^{y}(1-p(x_{k+1}r))^{k-y}\ \ \ 二項分布: B(k ,p(x_{k+1}, r))\]

\(k+1\)番目に選ばれた点のまわりの領域に入る、すなわち時刻\(k+1\)に結ばれるエッジの数の期待値\(l\)は、

\[\begin{align}l(r,k)&=<<yB(k, p(x_{k+1}, r))>_{y}>_{x_{k+1}}\\ &= <kp(x_{k+1}, r)>_{x_{k+1}} \\ &= k(-r^{2} + 2r) \end{align}\]

Remark: システムサイズ\(N\)との関連は無い

2. \(X_{i}\)を選ぶ確率が異なり、\(x\)を\([0,1]\)から一様に選ぶ場合

  • \(x\)の選び方は同じ
    \(\Rightarrow\)選ばれた点\(x\)によるネットワークは同じ
  • 時刻\(k\)までの次数の総和\(L\)

図. 時刻\(k\)までの次数の総和\(L\)

  • 緑の曲線: 理論式(1より)

    \[L = \frac{1}{2}(-r^{2} + 2r)k^{2}\]

    \(\Rightarrow\)一致

2. 過去の点を参照して次の点\(x\)が選ばれる

設定

  • 確率変数\(x\)は\(\Omega = [0, 1] \times [0,1]\)に一様分布する
  • \(X_{i}\)はそれぞれ予め\(S\)個の点をもつ
  • \(X_{i}\)ごとに、参照点(過去の点)から最も近い点を1つ選ぶ
  • 選ばれた点の参照点からの近さの順に\(X\)を整列する
  • \(X_{i}\)の順番に従い、確率\(P_{i}\)でそれぞれ点が選ばれる
  • 点が選ばれなかったとき(確率\(1-P_{i}\))は次の\(X_{j}\)について同様の操作

  • すべての\(X\)から点が選ばれなかったとき、その時刻には\(x_{0}\)が選ばれたとみなして、$時刻を1進め、同じ操作を行う

2-A. 過去に依らず点が選ばれる場合

  • 参照する点: なし
  • \(X_{i}\)が、\(N\)個の\(X\)の中で\(r+1\)(\(r = 1, 2, \cdots , n-1\))番目に選ばれたとき \(X_{i}\)まで順番が回ってくる確率は

\[p_{i}(r+1) = \frac{1}{_{n-1}C_{r}}\sum_{J = \langle j_{0}, \cdots ,j_{r-1} \rangle _{r}} \left( \prod_{j\in J}(1-P_{j}) \right) .\]

\(J = \langle j_{0}, j_{1}, \cdots ,j_{r-1} \rangle_{r}\)は、\(i\)を除く\(n-1\)個の要素から \(r\)個選んだときの組み合わせのうちの1揃いをあらわす

  • 1番目になるとき

\[p_{i}(1) = 1\]

ex)

\(N = 5\), \(i = 1\), \(r = 2\)のとき、\(X_{1}\)からの点が選ばれるまでに2つの\(X\)があり、その組み合わせは\((0,2)\), \((0,3)\), \((0,4)\), \((2,3)\), \((2,4)\), \((3,4)\)の6つであり、これら一つ一つの組が\(J\)である。この\(J\)に関して和をとり、組み合わせの数\(\ _{4}C_{2} = 6\)で割って期待値を求めている。

\[\begin{align} p_{i}(r+1) &= \left[(1-P_{0})(1-P_{2}) + (1-P_{0})(1-P_{3}) + (1-P_{0})(1-P_{4}) \right.\\ &\ \left. + (1-P_{2})(1-P_{3}) + (1-P_{2})(1-P_{4}) + (1-P_{3})(1-P_{4}) \right]/6 \end{align}\]

  • | 過去の点を参照しない | \(\Rightarrow\) \(X_{i}\)が\(r\)番目となる確率は等しい
  • \(X_{i}\)から点が選ばれる確率の期待値\(p_{i}\)

\[p_{i} = \frac{\sum_{r=1}^{n}p_{i}(r)}{n}P_{i}.\]

  • (\(S>K\)なら)毎時刻ごとにこの確率で\(X_{i}\)から点が選ばれることと等価

\(\Rightarrow\)単純な確率過程に帰着

  2. 過去の点を参照して次の点\(x\)を選ぶ

\[\begin{align}D(x, y) &= d(x,y)\\ &= \sqrt{(x_{1} - y_{1})^{2} + (x_{2} - y_{2})^{2} + \cdots + (x_{a} - y_{a})^{2}}\end{align}\]

\(X\)の数\(N\)に対する1ステップあたりの移動距離の平均値\(\phi\)

図8.

\(X\)の数\(N\)に対する1ステップあたりの移動距離の平均値\(\phi\)

  • 1ステップあたりの移動距離の平均値\(\phi\)

\[\phi \sim N^{-0.52}\]

1つの点のまわりの直径\(r\)による面積の期待値から、係数の絶対値は\(0.5\)より大きい

\[\frac{1}{2}r^{4} - \frac{3}{8}r^{3} + \pi r^{2} \sim r^{c},\ \ c < 2\ \ \ \text{when}\ \ 0<r<0.5\]

\[1 \sim r^{c}N \Rightarrow r \sim N^{-1/c},\ \ \ \frac{1}{c} > 0.5\]

1つの点を領域\(\Omega=[0,1]\times [0,1]\)から一様に選んだとき、
その点のまわりに長さ\(r\)によって作られる領域の面積の期待値

選んだ点が領域の中心付近では\(\mathcal{S} = \pi r^{2}\)とできるが、
領域\(\Omega\)の境界にかかっている場合には、それよりも小さい値となる。

図: 領域\(\Omega\)を分割した各領域

\[p(r)_{\Omega'} = \pi r^{2}\]

\[p(r)_{\Omega''_{i}} = i \sqrt{r^{2}-i^{2}} + r^{2} \left[ \pi -\arccos \frac{i}{r} \right]\]

\[\begin{align}p(r)_{\Omega''_{i,j}} = &\frac{1}{2}\left\{ \sqrt{r^{2}-i^{2}} + \min \left(j, \sqrt{r^{2}-i^{2}}\right) \right\}i \\ &+ \frac{1}{2}\left\{ \sqrt{r^{2}-j^{2}} + \min \left( i, \sqrt{r^{2}-j^{2}}\right) \right\}j \\ &+ \frac{1}{2}r^{2} \left\{ 2\pi -\arccos \frac{i}{r}-\arccos \frac{j}{r}\right. \\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \left. - \min \left( \frac{\pi}{2}, \arccos \frac{i}{r} +\arccos \frac{j}{r} \right) \right\} \end{align}\]

それぞれの領域で積分を実行し、その結果を足せば
任意の点を選んだときの面積の期待値が得られる。

\(X\)の要素数\(N\)と平均ステップ間距離との関係

図12. クラスター同士を結ぶモデルにおける、

\(X_{i}\)の数\(N\)と平均ステップ間距離\(\phi\)の関係

  • 平均ステップ間距離\(\phi\)は\(N\)に対して下に凸な関数系
  • \(N\)が小さいとき \(\Rightarrow\) 点の密度小 \(\rightarrow\) 点間距離増大(これまでと同じ理由)
  • \(N\)が大きいとき \(\Rightarrow\) 点の密度大 \(\rightarrow\) クラスター化促進 \(\rightarrow\) クラスター間距離増大

結論

  • 作成したそれぞれのモデルについて解析を行った

  • システムサイズに関してベキ的に変化するような量を観察することができた

    \[\phi \sim N^{-0.52}\]

  • 得られた結果は期待された結果と一致した
  • 系のサイズとの依存性について、基礎的な問題を解くことができた
